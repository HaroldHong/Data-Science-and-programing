- 课后
为让分析的网站类型单一化，我没有选择Google新闻，而是使用新浪财经板块的新闻。搜素黄金关键字，指定日期为8月1日，

一个csv文件中，再对每个url爬虫抓文字，真的是有点让人头大。而且不知道是不是xml2包里演算法的问题，经常url映射错误，但是
反复做几遍又能映射正确。资料抓取我花费了好几个小时....
